AMP AdSense Post Content Ad allow you to insert adSense ad inside content.

== Description ==

AMP AdSense Post Content Ad is a very simple plugin to let you insert adSense ad inside your amp post content. 
This plugin work as an add-on of "AMP for WordPress" plugin by Automattic or "AMP for WP â€“ Accelerated Mobile Pages" by Ahmed Kaludi, Mohammed Kaludi. That means, this will work only if any of those are in your Website.

You will be able to set the paragrapher number where you want to place your ad. May be after 2nd paragraph or 3rd ! up to you.

Till now, this will work only in WP Post Content. and Only one ad slot inside amp post content.


== Installation ==

1. Upload the folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. You may need to refresh your permalinks by going to `Settings > Permalinks` and tapping the `Save` button.
